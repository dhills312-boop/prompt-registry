# Prompt Registry Principles

1. Prompts guide workflows, not execution.
2. Prompts must have explicit roles and stop conditions.
3. Prompts avoid open-ended reasoning.
4. Prompts are model-agnostic by design.
5. Prompts should reduce retries and clarification loops.
6. Prompts should be understandable by humans without AI.
7. If a prompt grows too complex, it should be split.
8. Prompts are reviewed and refined over time.
