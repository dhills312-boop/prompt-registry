# Prompt Registry

This repository contains a curated set of prompts used for planning,
analysis, and review in software projects.

These prompts are:
- workflow-oriented, not generative
- bounded in scope
- reusable across models and tools
- designed to minimize unnecessary inference
- compatible with eco-conscious AI usage

This registry is the source of truth for how AI assistance is used.
Ad hoc prompting is discouraged when a registry prompt exists.

Prompts do not write production code.
They guide decision-making, structure work, and surface risks.
